# xCAT-docker-container-rocky9.4
This xCAT Docker image is built on Rocky Linux 9.6 and is fully supported for single-node xCAT management deployments. **With additional configuration, this image also supports a full xCAT High-Availability (HA) setup using: Docker Swarm, PCS (Pacemaker/Corosync) &amp; DRBD**
